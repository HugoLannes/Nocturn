Windows tray utility for multi-monitor setups

Black out any screen instantly. Keep the rest of your setup untouched.

Nocturn places a native fullscreen black overlay on the displays you choose, without disconnecting cables, moving windows, or changing your desktop topology.

  • Windows
  • OLED compatible
  • Native overlays

Why people use it

Built for calm desks, late sessions, and too many monitors.

Nocturn focuses on one thing: removing the screens you do not need right now, without disturbing the ones you still do.

How it works

Three calm steps.

The whole point is that it should feel obvious in less than a minute.

Step 1

Install the latest release

Grab the Windows installer from GitHub Releases and launch Nocturn from your tray.

Step 2

Pick the screens you want quiet

Use the display layout to darken only the monitors that are adding extra light.

Step 3

Wake everything instantly

Restore all displays in one action when you need the full setup again.
